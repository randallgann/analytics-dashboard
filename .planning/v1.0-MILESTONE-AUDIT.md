## Tech Debt

6 items across 3 areas. All non-blocking.

### Phase 1: Foundation + GitHub Pipeline
1. **INFRA-01 WAL implicit** — SQLite WAL mode works via sqlite3 gem 2.x default but has no explicit `pragmas: journal_mode: wal` in `database.yml`
2. **View-model coupling (GH-05)** — `index.html.erb` line 120 calls `GitHubMetric.latest_value("commit_frequency")` directly, bypassing the controller
3. **Dead helper** — `format_date` in `dashboard_helper.rb` is defined but never called from any view

### Phase 2: Social Feed
4. **Missing AuthError** — `RedditClient::AuthError` was removed when OAuth2 was abandoned for public JSON endpoint; no graceful auth error handling if credentials are re-added
5. **Dead error class** — `HnClient::RateLimitError` is defined but never raised by any `HnClient` method

### Scaffold
6. **Dead controller** — `hello_controller.js` (Rails scaffold artifact) is auto-loaded by Stimulus but never referenced in any view

### Total: 6 items
