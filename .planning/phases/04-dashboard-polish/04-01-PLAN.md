**GitHubMetric.delta_value** — Add class method to `app/models/github_metric.rb`:

```ruby
def self.delta_value(metric_type, days: 7)
  current = for_metric(metric_type).order(recorded_on: :desc).first&.value
  past    = for_metric(metric_type)
              .where(recorded_on: ...(Date.today - days))
              .order(recorded_on: :desc)
              .first&.value
  return nil if current.nil? || past.nil?
  current - past
end
```

Returns `nil` when fewer than 7 days of data exist (new deployment). The boundless range `...(Date.today - days)` selects records strictly before 7 days ago; the most recent of those is the baseline.

**SocialPost.ranked_by_engagement** — Add constant and class method to `app/models/social_post.rb`:

```ruby
GRAVITY = 1.8

def self.ranked_by_engagement(limit: 50)
  candidates = last_30_days.where.not(published_at: nil)
  candidates.sort_by do |post|
    age_hours  = [(Time.current - post.published_at) / 3600.0, 0].max
    engagement = post.score.to_f + (post.comment_count.to_f * 2)
    -(engagement / ((age_hours + 2)**GRAVITY))
  end.first(limit)
end
```

This uses the HN-style time-decay formula: `engagement / (age_hours + 2)^1.8`. Returns a plain Array (NOT ActiveRecord::Relation) — do NOT chain `.where` on the result. The controller must partition the result in Ruby. `comment_count * 2` weights discussion slightly higher than raw score.

**DashboardHelper additions** — Add `render_hero_metric` and `format_delta` to `app/helpers/dashboard_helper.rb`:

```ruby
def render_hero_metric(label, value, delta, period = "7d")
  content_tag(:div, class: "dash-hero-card") do
    label_el = content_tag(:div, label, class: "dash-hero-label")
    value_el = content_tag(:div, format_number(value), class: "dash-hero-value")
    delta_el = format_delta(delta, period)
    safe_join([label_el, value_el, delta_el])
  end
end

def format_delta(delta, period = "7d")
  return content_tag(:div, "No baseline yet", class: "dash-delta dash-delta--neutral") if delta.nil?

  sign = delta >= 0 ? "+" : ""
  css  = delta > 0 ? "positive" : (delta < 0 ? "negative" : "neutral")
  text = "#{sign}#{format_number(delta.to_i)} this #{period}"
  content_tag(:div, text, class: "dash-delta dash-delta--#{css}")
end
```

`format_delta` handles nil gracefully (shows "No baseline yet" in neutral color). Uses existing `format_number` for comma-delimited display.
