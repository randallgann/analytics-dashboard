<task type="auto">
  <name>Task 2: Create GithubClient service with typed errors and stats retry</name>
  <files>
    app/services/github_client.rb
    test/services/github_client_test.rb
  </files>
  <action>
    1. Create `app/services/` directory if it does not exist.

    2. Create `app/services/github_client.rb` following the research code example closely:
       - Define inner error classes: `RateLimitError < StandardError`, `AuthError < StandardError`, `NotFoundError < StandardError`, `StatsUnavailableError < StandardError`
       - `REPO_SLUG` constant: `Rails.application.credentials.dig(:github, :repo_slug) || ENV.fetch("GITHUB_REPO_SLUG", "openclaw/openclaw")`
       - `initialize`: Create `Octokit::Client.new(access_token:)` using `Rails.application.credentials.dig(:github, :token) || ENV["GITHUB_TOKEN"]`
       - Public methods wrapping Octokit with error mapping:
         - `repository` — calls `@client.repository(REPO_SLUG)`, rescues `Octokit::Unauthorized` -> `AuthError`, `Octokit::NotFound` -> `NotFoundError`, `Octokit::TooManyRequests` -> `RateLimitError`
         - `commit_activity_stats` — calls `fetch_stats { @client.commit_activity_stats(REPO_SLUG) }`
         - `contributors_stats` — calls `fetch_stats { @client.contributors_stats(REPO_SLUG) }`
         - `open_issues_count` — `@client.search_issues("repo:#{REPO_SLUG} is:issue is:open").total_count` (uses Search API to exclude PRs per Pitfall 1)
         - `closed_issues_count` — `@client.search_issues("repo:#{REPO_SLUG} is:issue is:closed").total_count`
         - `pull_requests_count(state:)` — `@client.search_issues("repo:#{REPO_SLUG} is:pr is:#{state}").total_count`
         - `latest_release` — `@client.latest_release(REPO_SLUG)` rescuing `Octokit::NotFound` -> nil
         - `releases` — `@client.releases(REPO_SLUG)` for release cadence data
       - Private `fetch_stats` method: retry loop (max 3 retries, exponential backoff `sleep(2 ** retries)`) that re-calls block when result is nil/empty (GitHub 202 response). Raises `StatsUnavailableError` after 3 retries.

    3. Create `test/services/github_client_test.rb`:
       - Test that `initialize` creates an Octokit client (mock/stub)
       - Test that `repository` maps `Octokit::Unauthorized` to `AuthError`
       - Test that `repository` maps `Octokit::TooManyRequests` to `RateLimitError`
       - Test that `fetch_stats` retries on empty response up to 3 times
       - Test that `latest_release` returns nil when repo has no releases

    NOTE: The `test/services/` directory will need to be created. Rails autoloads `app/services/` via Zeitwerk, so no explicit require needed.
  </action>
